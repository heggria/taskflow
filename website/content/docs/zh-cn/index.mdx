title: taskflow 文档
description: 从 taskflow 0.2 开始，运行第一张可验证任务图，或直接进入编译器 / 运行时参考。
---

taskflow 是面向 coding-agent 子代理的声明式编排运行时。你定义一张图，在花 token 前验证它，让每个阶段隔离执行，并且只把最终结果返回宿主对话。

在 **0.2** 中，这张图还成为了一份编译合同：可以用 TypeScript 编写、检查 FlowIR、replay 已记录的运行决策，并且只重算因输入变化而 stale 的阶段。

## Compiler / Runtime 路径

<Steps>
  <Step>
    **花 token 前先验证。** 从 `/tf verify` 或 `taskflow_verify` 开始，结构错误零 token。
  </Step>
  <Step>
    **编译执行合同。** 使用 `/tf ir` 检查 FlowIR 与内容哈希。
  </Step>
  <Step>
    **有意地区分 resume 与 replay。** Resume 继续未完成工作；replay 只在零 token 下重判已完成轨迹。
  </Step>
  <Step>
    **只重算 stale frontier。** 应用最小重算之前，先运行 `why-stale`。
  </Step>
</Steps>
