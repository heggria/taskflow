taskflow is a declarative orchestration runtime for coding-agent subagents. You define a graph, verify it before token spend, execute each phase in isolation, and return only the final result to the host conversation.

In **0.2**, that graph also becomes a compiled contract: author with TypeScript, inspect FlowIR, replay recorded decisions, and recompute only the phases made stale by a change.

## The compiler/runtime path

<Steps>
  <Step>
    **Verify before spend.** Start with `/tf verify` or `taskflow_verify`; structural errors cost zero tokens.
  </Step>
  <Step>
    **Compile the contract.** Inspect FlowIR and the content hash with `/tf ir`.
  </Step>
  <Step>
    **Resume or replay intentionally.** Resume unfinished work; replay a finished trace only for zero-token what-if decisions.
  </Step>
  <Step>
    **Recompute the stale frontier.** Use `why-stale` before applying a minimal recompute.
  </Step>
</Steps>
